# Mapas mentais

Coloque nesta pasta os mapas mentais finais do projeto.

Formato recomendado:

- `organizacao-sistema-nervoso.png`
- `medula-espinhal.png`
- `tronco-encefalico.png`
- `cerebelo.png`
- `diencefalo.png`
- `telencefalo.png`
- `sistema-limbico.png`
- `nervos-cranianos.png`

Os arquivos `.svg` incluídos aqui são exemplos temporários para a versão MVP. Você pode substituí-los pelos mapas criados no Gemini mantendo o mesmo nome do arquivo, ou adicionar novos mapas e registrar no bloco `mapLibrary` do `index.html`.
